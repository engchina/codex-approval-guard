# AGENTS.md

## プロジェクト言語

- このプロジェクトの公式言語は日本語です。
- ドキュメント、README、コードコメント、UI文言、エラーメッセージは原則として日本語で記述します。
- 技術用語、API名、ファイル名、識別子、外部サービス名は既存の正式表記を優先し、無理に翻訳しません。
- ユーザーとの会話は、ユーザーが使用した言語に合わせます。

## 開発方針

- このプロジェクトは Codex App 向けの開発プロジェクトです。
- 既存の構成、命名、UI表現、実装パターンを先に確認してから変更します。
- 変更は目的に対して必要最小限にとどめ、無関係なリファクタリングや表記揺れの修正を混ぜません。
- コメントを追加する場合は、実装意図や注意点を補う内容に限定し、コードを読めば分かる説明は避けます。

## UI/UX 設計

- UI/UX の設計、実装、改善、レビュー、修正を行う場合は、必ず `ui-ux-pro-max` skill を使用します。
- 着手前に `ui-ux-pro-max` の `SKILL.md` を確認し、必要に応じて `--design-system` によるデザインシステム生成や stack guideline を参照してから方針を決めます。

## 検証

- 実装変更後は、変更範囲に合った lint、test、build、または手動確認を行います。
- 検証できなかった場合は、理由と残るリスクを明確に報告します。
- 既存の未コミット変更がある場合は、ユーザーの作業として扱い、明示的な依頼なしに戻しません。
